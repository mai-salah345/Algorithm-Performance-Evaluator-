# 🚀 Algorithm Performance Evaluator

The **Algorithm Performance Evaluator** is a professional desktop application designed to bridge the gap between theoretical Big O analysis and practical execution. This tool enables students and developers to analyze the efficiency of custom algorithms by testing them against varying input sizes and automatically estimating their complexity classes.

## 🎯 Project Overview
The core purpose of this application is to provide a dedicated environment where users can define algorithms, run them against real data, and see the mathematical complexity in action through visual feedback.

### ✨ Key Features
* **Automated Complexity Estimation:** Automatically determines the closest complexity class, such as $O(1)$, $O(n)$, $O(n \log n)$, $O(n^2)$, $O(n^3)$, and $2^n$.
* **Performance Visualization:** Generates real-time visual charts plotting **Execution Time vs. Input Size** to provide a clear performance overview.
* **Comprehensive Testing Environments:**
    * **Manual Mode:** For debugging and verifying specific, user-controlled test cases.
    * **Auto Mode:** For stress-testing algorithms using automatically generated **Best, Worst, and Average** cases.
